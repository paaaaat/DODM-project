# DODM-project
Il repo del progetto del corso di Discrete Optimization and Decision Making.





## Setup

### 1.Assicurati di aver installato *Rust* e *Cargo*:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 2. Installa *Typst*
```bash
cargo install --locked typst-cli
```
